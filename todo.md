# MedManager-PRO - Sistema de Gestão Farmacêutica
## Plano de Implementação MVP

### Arquivos Principais a Criar:

1. **src/pages/Dashboard.tsx** - Dashboard principal com métricas
2. **src/pages/Login.tsx** - Tela de login dual (email/CNPJ)
3. **src/pages/Products.tsx** - Gestão de produtos farmacêuticos
4. **src/pages/Inventory.tsx** - Controle de estoque
5. **src/pages/Orders.tsx** - Gestão de pedidos
6. **src/pages/Compliance.tsx** - Conformidade ANVISA
7. **src/components/Layout/Sidebar.tsx** - Navegação lateral
8. **src/components/Layout/Header.tsx** - Cabeçalho com notificações

### Funcionalidades MVP:
- ✅ Interface de login com tema farmacêutico
- ✅ Dashboard com cards de métricas principais
- ✅ Gestão básica de produtos (CRUD)
- ✅ Controle de estoque com alertas
- ✅ Sistema de pedidos simplificado
- ✅ Painel de conformidade ANVISA
- ✅ Layout responsivo em PT-BR

### Tema de Cores (Farmacêutico):
- Primary: Azul médico (#0066CC)
- Secondary: Verde saúde (#00AA44)
- Accent: Branco clínico (#FFFFFF)
- Warning: Laranja alerta (#FF8800)
- Error: Vermelho urgência (#CC0000)

### Componentes de UI:
- Cards informativos com ícones médicos
- Tabelas de dados com filtros
- Formulários de cadastro
- Alertas de conformidade
- Gráficos de métricas
